# Travilla User Panel

## clone the repo
```bash
git clone https://github.com/Ahmed-Elgalay-Projects-Collaborations/travilla-user-panel.git
```

## Run with npm
```bash
npm i
npm run dev
```

## Run with Docker
```bash
docker-compose up --build
```
